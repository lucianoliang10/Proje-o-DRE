# Projeção Anual (2026-2036)

App em Streamlit para projetar o P&L anual de 2026 até 2036, aplicando percentuais apenas nas linhas de nível 2.

## Como executar

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Estrutura

- `app.py`: aplicação principal do Streamlit.
- `requirements.txt`: dependências.
